class kraken.core.builder.Builder¶

Bases: object

Builder object for building objects in DCC’s. Sub-class per DCC in a plugin.

Arguments: kSceneItem – Object, kraken scene item that you want to pair. dccSceneItem – Object, dcc scene item that you want to pair.

Return: True if successful.
