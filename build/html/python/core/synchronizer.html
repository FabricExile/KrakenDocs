Synchronizer¶

Contents:

class kraken.core.synchronizer.Synchronizer(target=None)¶

Bases: object

The Synchronizer is a singleton object used to synchronize data between Kraken objects and the DCC objects.

__init__(target=None)¶

Initializes Synchronizer.

The target object’s hierarchy will be traversed and synchronized.

Arguments: target – Object, top Kraken object to synchronize.

getTarget()¶

Gets the target object of the synchronizer.

Return: Object, the object that is the target of synchronization.

setTarget(target)¶

Sets the target for synchronization.

Arguments: target – Object, top Kraken object to synchronize.

Return: True if successful.

getHierarchyMap()¶

Gets the hierarchy map from the Inspector.

Return: Dict, the hierarhcy map. None if it hasn’t been created.

createHierarchyMap(kObject)¶
clearHierarchyMap()¶

Clears the hierarhcy map data.

Return: True if successful.

sync()¶

Synchronizes the target hierarchy with the matching objects in the DCC.

Return: True if successful.

synchronize(kObject)¶

Iteration method that traverses the hierarchy and syncs the different object types.

Arguments: kObject – Object, object to synchronize.

Return: True if successful.

getDCCItem(kObject)¶

Gets the DCC Item from the full decorated path.

  • This should be re-implemented in the sub-classed synchronizer for each

plugin.

Arguments: kObject – object, the Kraken Python object that we must find the corresponding DCC item.

Return: DCC Object, None if it isn’t found.

syncXfo(kObject)¶

Syncs the xfo from the DCC objec to the Kraken object.

  • This should be re-implemented in the sub-classed synchronizer for each

plugin.

Arguments: kObject – Object, object to sync the xfo for.

Return: True if successful.

syncAttribute(kObject)¶

Syncs the attribute value from the DCC objec to the Kraken object.

  • This should be re-implemented in the sub-classed synchronizer for each

plugin.

Arguments: kObject – Object, object to sync the attribute value for.

Return: True if successful.
