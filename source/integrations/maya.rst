================
Maya Integration
================

The Maya integration is composed of the Kraken plug-in for Maya but also the Maya plug-in that adds a menu entry for launching the Kraken UI.


Sections
========

.. contents:: Usage Documentation
   :local:


Maya Plug-in
------------

To enable the Maya plug-in you simply need to add the following path to your :envvar: MAYA_MODULE_PATH variable.

`*/kraken/DCCIntegrations/maya`
